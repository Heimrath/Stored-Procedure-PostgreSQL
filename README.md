# Stored Procedure (PostgreSQL) — Class Exercises (Neon + Google Colab)

This repository contains a short class assignment about **PostgreSQL stored procedures/functions**, developed in **Google Colab** and executed on **Neon (PostgreSQL)**.  
The goal is to solve a set of exercises proposed by the professor using SQL routines (functions/procedures) and basic database operations.

## Tech Stack

- **PostgreSQL**: 17.8  
- **Environment**: Google Colab (notebook-based)
- **Database hosting**: Neon

> Version info used during development:
> PostgreSQL 17.8 (a48d9ca) / System Version 17.8 (a48d9ca)

## What you’ll find here

- SQL scripts and/or notebook code used to:
  - connect from Google Colab to a Neon PostgreSQL database
  - create tables and sample data
  - implement stored procedures/functions to solve the proposed exercises
  - run queries to validate results

## How to run

1. Create a PostgreSQL database on **Neon**.
2. Open the project notebook in **Google Colab**.
3. Set your Neon connection string/credentials in the notebook.
4. Run the cells to:
   - connect to the database
   - execute the SQL (schema + routines)
   - execute the exercise queries

## Notes

- This is a learning project for database programming with routines (stored procedures/functions).
- The exercises and solutions are intentionally kept simple and focused on practice.
